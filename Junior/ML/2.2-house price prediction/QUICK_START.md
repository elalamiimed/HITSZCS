# 🚀 Quick Start Guide

## 1. Installation (30 seconds)

```bash
# Install dependencies
pip install -r requirements.txt
```

## 2. Quick Demo (1 minute)

```bash
# Run quick demonstration
python demo.py
```

## 3. Full Analysis (5 minutes)

```bash
# Run complete analysis
python house_price_prediction.py
```

## 4. Advanced Models (3 minutes)

```bash
# Run advanced regression models
python advanced_regression_models.py
```

## 5. Interactive Visualizations (2 minutes)

```bash
# Run interactive visualizations
python interactive_visualization.py
```

## 6. Jupyter Notebook (Open in browser)

```bash
# Start Jupyter notebook
jupyter notebook house_price_analysis.ipynb
```

## 🛠️ Troubleshooting

### Common Issues

1. **Import Error**: Run `pip install -r requirements.txt`
2. **Memory Error**: Use `demo.py` instead of full scripts
3. **Plot Display**: Ensure matplotlib backend is configured

### System Requirements

- **Python**: 3.7+
- **RAM**: 4GB minimum
- **Storage**: 500MB
- **OS**: Windows/Linux/macOS

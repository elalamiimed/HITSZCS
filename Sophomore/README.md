- **Difficulty Level:** Moderate (significantly easier than Physics IA)

   - Amplitude, frequency, wavelength
   - Reflection and refraction
   - Refractive index applications
   - Intensity calculations
   - Doppler effect
   - Diffraction phenomena
   - Interference patterns
   - Single and double slit experiments

- Tutorial sessions with guided practice

#### Data Structures

**Course Details:**
- **Official Title:** Data Structures and Algorithms
- **Focus:** Algorithm design and implementation
- **Prerequisite:** Strong C++ programming skills (from High-Level Language Programming)

**Data Structures Covered:**
- All C++ data types
- 2D arrays and array operations
- Stacks and queues
- Linked lists and variations
- Tree structures
- Other advanced data structures

**Algorithms Covered:**

*Sorting:*
- Bubble sort
- Insertion sort
- Selection sort
- Advanced sorting algorithms

*Array Operations:*
- 1D and 2D array traversal
- Array searching techniques
- Array validation

*String & Pattern Matching:*
- Substring search
- KMP (Knuth-Morris-Pratt) Algorithm
- Pattern matching techniques

*Linked List Algorithms:*
- Linked list traversal
- List reversal
- Finding intersections
- List manipulation techniques

**Course Structure:**
- Organized slide presentations
- Programming assignments and projects
- Algorithm implementation practice
- Problem-solving exercises

**Tips for Success:**
- Build on solid C++ foundation from previous course
- Practice implementing each data structure from scratch
- Understand algorithm complexity and efficiency
- Test implementations with various input cases
